A SOLID ECOMMERCE PLATFORM

Kommerce is a full featured e-commerce platform adhering to SOLID (clean code) principles. It's framework agnostic, themable and extendable. It ships with 100% code coverage and acceptance tests. Kommerce has its own admin panel making it easy to create products, coupons, manage orders and more.

SOLID

Starting from a clean slate using clean coding principles Kommerce is the culmination of years of effort to ditch the must-have-framework mindset and embrace software craftsmanship in every line of source code to create a truly solid ecommerce platform.

Themeable

Extend, reuse, create. Leverage existing themes and inherit things you like or customize just the pieces you'd like in your own theme based on another theme. The goal is to increase efficiency and usability without always starting from scratch but you can do that too. Themes are flexible, easy to use, and simple to understand.

Extensible

Need things to behaviour differently? Want something to happen on an event? No problem. There's already systems in place for that just extend or observe and you're off. Modify what's necessary without ever touching core keeping the separation of concerns complete and your business rules where they belong: in your store.

Install Kommerce

Install with Docker


$ docker run -d -p 9000:9000 inklabs/kommerce-laravel-php71-fpm)
$ open localhost:9000

Install with Composer


$ git clone https://github.com/inklabs/kommerce-laravel.git
$ cd kommerce-laravel
$ composer install
$ composer create-project
$ php bin/initialize-test-db.php
