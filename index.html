A SOLID ECOMMERCE PLATFORM

Kommerce is a full featured e-commerce platform adhering to SOLID (clean code) principles. It's framework agnostic, ships with 100% code coverage and acceptance tests. Kommerce has its own admin panel making it easy to create products, coupon codes, manage orders and more.

Getting Started

Using Docker

$ docker pull kommerce/latest
$ docker run kommerce/latest
$ open http://localhost:3000

Using Composer

$ composer install inklabs/kommerce-core
